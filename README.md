# 查寝系统访问跳转页面

## 说明

本仓库用于存储查寝系统的动态 IP 跳转页面。

## 功能

- 自动检测校园网 IP 变化
- 提供网络环境检测（校园网 WiFi vs 流量）
- 自动跳转到查寝系统

## 访问地址

> 注意：Gitee Pages 服务已停止，当前使用其他平台部署

## 使用方法

1. 连接校园网 WiFi
2. 访问上方 Gitee Pages 地址
3. 页面会自动检测并跳转到查寝系统

## 技术实现

- IP 检测：Python 脚本定时检测本机 IP
- 自动推送：Git 自动提交并推送到 Gitee
- 前端检测：JavaScript 检测网络连通性

## 注意事项

- 仅在校园网内可访问（内网 IP：10.x.x.x）
- 使用流量会显示提示
- IP 变化后会自动更新（每天 21:00 或手动触发）
- 本仓库仅存储静态跳转页面，不包含后端代码
